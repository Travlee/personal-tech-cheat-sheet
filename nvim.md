# CMake Tools - Plugin
`:CMakeLaunchArgs --arg1 --arg2 arg3`: Handy to pass args to CMakeRun command
`<leader>cr`: Runs CMakeRun command


# Normal Mode

## Jumps
`Ctrl + O`: Jump backwards in jump list.
`Ctrl + I, <TAB>`: Jump forwards in jump list.
`:jumps`: Shows jump history.

## Movement
`Ctrl + u, Ctrl + d`: Scroll half a screen up and down, respectively.
`zz`: Center screen on current line.
`f{char}, F{char}`: Search forwards and backwards, respectively, on the current line to next or previous occurrence of the {char}.
`;`: Repeats the last search forward.
`,': Repeats the last search backwards.
`t{char}, T{char}`: Similiar to `f` but places cursor on peceding character.
